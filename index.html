Project Overview

Designed, rigged, and animated a mascot character in Rive to represent my personal brand. The goal was to create a fun, approachable, and professional character that communicates my creative style while being technically reusable for my portfolio website.

My Role

Solo Designer & Animator (Concept, Rigging, Animation, Branding)

Technologies Used

  • Rive (Rigging & Animation)
  • Figma (Design & Iterations)
  • Vector Illustration

Key Features

  • Round, friendly mascot designed through silhouette and color exploration
  • Rigged with a clean bone hierarchy for smooth movement
  • 3 interactive animations (idle belly tap, greeting wave, dance/jig)
  • State machine setup for event-based triggers (on_hover, on_click)
